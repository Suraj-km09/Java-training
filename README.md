# Problems on Loop

## Basic Loop Understanding

1. Write a Java program that prints numbers from 1 to N using a loop, but skip numbers that are multiples of 3 and print the remaining numbers in a list.

2. Write a Java program that prints numbers from N down to 1, but stop printing if the number becomes divisible by 7.

3. Write a Java program to print all even numbers between 1 and N and also return the count of even numbers.

4. Write a Java program that prints all odd numbers from 1 to N and calculates their total sum.

5. Write a Java program that prints the multiplication table of a given number up to 10 and stores the results in an array.

6. Write a Java program to print all numbers between 1–100 that are divisible by both 3 and 5 and return their count.

7. Write a Java program to count how many numbers between 1 and N are divisible by 2 or 5.

8. Write a Java program to print numbers from 1 to N but skip numbers that are multiples of 4 using continue.

9. Write a Java program to print numbers from 1 to 100 but skip 50 and also print how many numbers were printed.

10. Write a Java program to print numbers from N to 1 and also store them in a reversed array.
